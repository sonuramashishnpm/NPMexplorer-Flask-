📘 NPM Explorer - Project Documentation

🧠 Introduction

NPM Explorer is an all-in-one tech platform built for students, developers, and AI enthusiasts. It integrates education resources, live stock analysis, news updates, and smart AI tools — in one powerful web interface.

🎯 Purpose

This project aims to provide a comprehensive learning and information hub, making it easier for users to access educational content, stay updated with the latest news, analyze stock market trends, and interact with AI-driven chatbots.

💡 Features

  • Education Tab: Class 8–12 subjects, AI-generated PDFs and certification.
  • News Section: Latest technology and world news via NewsAPI.
  • Stock Market: Real-time data with charts and AI predictions.
  • AI Chatbot: Powered by GPT-based tech for interactive Q&A.
  • Login/Signup System: User authentication with data saving.

🖥 Layout Overview

  • Header: Logo + Navigation Menu
  • Main Tabs: Education | News | Stocks | AI Chat
  • Dynamic Panel: Shows content based on selected tab
  • Footer: External links and creator credit

⚙️ Technologies Used

  • HTML5 & CSS3 for structure & styling
  • JavaScript for tab switching, API calls
  • Netlify for live hosting
  • yFinance API, NewsAPI, Wikipedia API
  • SQLite used for storing user data locally

🔗 API Integration

  • Stock API: https://query1.finance.yahoo.com/v7/finance/chart/{symbol}
  • News API: https://newsapi.org/v2/top-headlines?country=in
  • Wikipedia Summary: https://en.wikipedia.org/api/rest_v1/page/summary/{query}

🚀 Deployment Guide (Netlify)

  1. Push your code to GitHub
  2. Login to Netlify and connect your GitHub repo
  3. Set Build Command: npm run build (if using a bundler)
  4. Set Publish Directory: dist or build
  5. Deploy and share your live link!
